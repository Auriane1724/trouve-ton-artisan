# Trouve ton artisan

Application web permettant d’afficher des artisans et leurs métiers à partir d’une base de données MySQL.
Frontend en React (Vite) + Backend en Node.js/Express + ORM Sequelize.

## Fonctionnalités

- Affichage de la liste des artisans
- Affichage des métiers associés (relation N–N)
- API REST : `GET /artisans`

## Prérequis

- Node.js (recommandé : v18+)
- MySQL (installé via Homebrew ou autre)
- npm
